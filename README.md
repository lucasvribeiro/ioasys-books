# Welcome to ioasys Books 📖

A web application to show books informations.

### [🌍 DEMO](https://ioasysbookss.netlify.app/)

### 📚 Stack Used:

- React 17.0.2
- npm 8.3.0

### 🏃 Running Steps:

1. `git clone https://github.com/lucasvribeiro/ioasys-books.git`
2. `cd ioasys-books`
3. `npm i` or `npm install`
4. `npm start`
